# UX Gaps

## Top 5 problemi
1. Focus states quasi assenti: molti input/button non hanno focus visibile -> accessibilita e keyboard navigation deboli.
2. Navigazione da tastiera povera: solo hotkey globale per quick add; manca command palette e shortcut per viste.
3. Vista Mese non e' un calendario/heatmap: gap e giorni mancanti non saltano all'occhio.
4. Tipografia non garantita: font “Source Sans 3/Sora” non bundlati, risultato variabile per OS.
5. Metadati attivita poco visibili: tag/risorsa non mostrati nelle liste principali, verbali senza vista dedicata.

## 5 miglioramenti ad alto impatto
1. Aggiungere focus/hover/selected coerenti (outline, background, contrasto) su tutti i controlli.
2. Command palette + shortcut per passare tra Giorno/Settimana/Mese/Ricerca (Ctrl+1/2/3/4) e quick add.
3. Vista Mese con calendario a griglia e heatmap di gap (giorni vuoti evidenziati, weekend distinti).
4. Validazioni inline nei form (min/max minuti, titolo obbligatorio, warning su durate anomale).
5. Mostrare tag/risorsa/verbale come chip nelle liste + filtri rapidi per status/cliente/tag.

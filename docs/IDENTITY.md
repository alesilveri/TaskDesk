# TaskDesk Identity

## Name
TaskDesk

## Tagline
Il tuo registro attivita smart, al posto di Excel.

## UX principles
- Minimale e veloce: zero frizioni, azioni principali in 1-2 click.
- Focus Giorno/Settimana/Mese con totali e gap sempre visibili.
- Inserimento rapido + autocomplete clienti + recenti.
- Affidabile e credibile: suggerimenti realistici, nessuna invenzione di attivita.

## Visual direction
- Pulita, professionale, con accenti caldi per evidenziare gap/avanzamento.
- Contrasti alti in light/dark, stati hover/selected sempre leggibili.

## Palette (base)
Light:
- Background: #F7F4EF
- Surface: #FFFFFF
- Text: #0B0F14
- Muted: #6B7280
- Accent Teal: #2BB3A3
- Accent Amber (gap): #F59E0B
- Danger: #E3514D

Dark:
- Background: #0B0F14
- Surface: #141A22
- Text: #F4F6F8
- Muted: #9AA3AF
- Accent Teal: #38C7B6
- Accent Amber (gap): #F6B44B
- Danger: #FF6B6B

## Typography
- Headings: Sora (600-700)
- Body: Source Sans 3 (400-600)
- Numbers/metrics: JetBrains Mono (500)

## Icon idea
- Squircle with a subtle grid and a check/desk line, to convey task + desk + precision.

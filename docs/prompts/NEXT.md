# TaskDesk - Next Tasks Prompt

Contesto: TaskDesk e' un desktop tracker per ricostruire attivita da inserire nel gestore commesse. Non e' un tool di fatturazione.

Obiettivo: definire la prossima milestone e produrre un piano di lavoro breve.

Istruzioni:
- Leggi `docs/DECISIONS.md` e `docs/INSPIRATION.md`.
- Verifica backlog e stato in `CHANGELOG.md` e `README.md`.
- Proponi 3-5 task concreti (max 2 giorni di lavoro ciascuno).
- Aggiorna `docs/DECISIONS.md` se fai scelte nuove.

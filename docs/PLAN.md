# TaskDesk Plan

Aggiornato: 2026-01-20

## MVP (stabilità + utilità)
- UI day/week/month pulita e veloce con target/gap chiari.
- Export XLSX (Daily Task ICON) + copia Gestore (mensile/settimanale).
- Backup/restore con rotazione affidabile.
- Settings per target, tema, giorni lavorativi.

## Next (qualità prodotto)
- Screenshot reali + social preview + branding coerente.
- Smart suggestions credibili (micro-attività solo se gap reale).
- CI su push/PR + dependabot con policy LTS.
- Smoke test automatico (DB temp + export verificato).

## Later (evoluzione)
- Deep links (taskdesk://today, /month/2026-01).
- Notifiche smart configurabili.
- Import CSV clienti guidato + merge/dedup evoluti.

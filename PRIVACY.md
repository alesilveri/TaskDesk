# Privacy

TaskDesk is a local-first desktop app. All data is stored locally on your machine and is not transmitted to any external service.

## Data stored locally
- Attivita, clienti, e riepiloghi
- Preferenze utente e impostazioni

## Network
- No analytics
- No external tracking
- Optional integrations are off by default

## AI (optional)
If enabled in the future, AI features will be opt-in and require explicit configuration of the provider and API key.
